# 🌍 TravelGo

TravelGo es una aplicación web básica para explorar destinos turísticos, guardarlos como favoritos y realizar reservas de manera simple.

---

## 🚀 Funcionalidades

- 🔍 Búsqueda de destinos
- ⭐ Guardar destinos favoritos
- 🧾 Reservar destinos con control de cupos
- 💰 Cotizador de viajes
- 📱 Interfaz moderna con HTML, CSS y JavaScript

---

## 📂 Estructura del proyecto


---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📸 Vistas principales

### 🏠 Inicio
- Barra de navegación
- Hero principal
- Buscador de destinos
- Recomendaciones

### 📍 Destinos
- Lista de destinos disponibles
- Sistema de reservas con cupos dinámicos
